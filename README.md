# Mocha

## Goals

- [ ] Scanner
- [ ] Parser
  - [ ] Syntax verification
  - [ ] Type checking
  - [ ] IR code generation
- [ ] Instruction
  - [ ] Selection
  - [ ] Scheduling
- [ ] Register allocation
- [ ] Multi-platform support
- [ ] Self-hosted

## Building and Compiling

Building

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
```

Compiling a file

```bash
$ ./mocha file_name.moc [OPTIONS]

```
