# Mocha Language Description

## Function Definition
```
func func_name(arg1: arg1_type, arg2: arg2_type, ...) => ret_type {
    function body
    
    ret ret_val;
}
```

## Variable Definition
```
var var_name: var_type;
const var_name: var_type = var_value;
```

## Arithmetic Operators
    - ```+```: Addition
    - ```-```: Subtraction
    - ```*```: Multiplication
    - ```/```: Division
    - ```%```: Modulo
    - ```**```: Exponential
